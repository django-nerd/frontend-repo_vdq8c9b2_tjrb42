Sobre o Projeto

Este website demonstra o uso criativo de HTML, CSS e JavaScript moderno. Inclui animações scroll-based, cursor personalizado, efeitos parallax e um objeto 3D interativo.

Créditos

GSAP, ScrollTrigger, Locomotive Scroll, VanillaTilt e Spline ajudam a tornar a experiência imersiva.
